API de Envio de Mensagens para WhatsApp

Esta é uma API simples para enviar mensagens para o WhatsApp usando a API.

Pré-requisitos
Antes de começar, você precisa ter as seguintes coisas configuradas:

- Um número de telefone do WhatsApp registrado na sua conta WhatsApp.
